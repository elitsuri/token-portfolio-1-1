# Token Portfolio 1

> Crypto portfolio tracker with on-chain data and analytics

## ✨ Features
- User authentication with JWT
- CRUD operations for main resources
- RESTful API with proper status codes
- Database migrations and seed data
- Docker containerization

## 🧰 Tech Stack
Go, Ethereum, PostgreSQL, Docker

## 🏗️ Architecture
Backend service with Go, frontend user interface, and database persistence

## 🚀 Quick Start

### Prerequisites
- Docker & Docker Compose
- SQLite / PostgreSQL

### Setup

```bash
# Clone the repository
git clone https://github.com/elitsuri/token-portfolio-1
cd token-portfolio-1

# Copy environment variables
cp .env.example .env
```

### Run

```bash
docker compose up --build
```
